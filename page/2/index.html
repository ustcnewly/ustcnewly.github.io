Different Ways of Injecting Latent Code

Posted on 2020-05-06 | In paper note

Given latent code z, how to inject it into an auto-encoder. skip-yes/no means whether using skip-connection is possible.

  1. Append it to the input image [1]* or add it to the input image [7] (skip-yes)

  2. Append it to each layer in encoder [2] (skip-yes)

  3. Add it to the bottleneck feature [3] (skip-no)

  4. Add to decoder layers using AdaIn [4]* (skip-no) or a variant of AdaIn [5] (skip-yes)

  5. Add to decoder layers without AdaIn [6] [8]*

Injecting structured latent code [9]
